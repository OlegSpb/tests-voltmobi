== README

* Ruby 3.2.1

* SQLite

* Админ и первый юзер создаются из <tt>rake db:seed</tt>.

* Заполнить БД тестовыми данными: <tt>rake populate:all</tt>.

* Деплой через capistrano.

* Деплой к себе сделан только как Staging.

* Код покрыт функциональными тестами. Делал их впервые, поэтому зачитался документации.
